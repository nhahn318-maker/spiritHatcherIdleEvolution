# Spirit Hatchers - Test Checklist

## 1. Save/Load Test

- Start new game
- Add resources
- Close Play Mode
- Open Play Mode again
- Check if resources are saved

## 2. Hatch Test

- Add 10 Egg Tickets
- Hatch 10 times
- Check if creatures are added
- Check if duplicates give rewards
- Check if resources decrease correctly

## 3. Collection Test

- Open Collection screen
- Owned creatures should be visible
- Locked creatures should show ???
- Click owned creature
- Creature Detail should open

## 4. Upgrade Test

- Add Coin
- Upgrade creature
- Level should increase
- Coin should decrease
- Power should update
- Save/load should preserve level

## 5. Evolution Test

- Set creature to required level
- Add required Food and Crystal
- Press Evolve
- Form should change
- Sprite should update
- Resources should decrease
- Final form should disable Evolve button

## 6. Expedition Test

- Select creature
- Start expedition
- Wait until complete
- Claim reward
- Resources should increase
- Creature should become available again

## 7. Offline Expedition Test

- Start short test expedition
- Stop Play Mode
- Wait
- Start Play Mode again
- Expedition should be complete

## 8. Daily Reward Test

- Claim daily reward
- Button should disable
- Save/load should remember claimed status

## 9. UI Navigation Test

- Home to Hatch
- Home to Collection
- Home to Expedition
- Back to Home
- No overlapping screens

## 10. Android Test

- Build APK or AAB
- Install on phone
- Check portrait orientation
- Check UI scaling
- Check save/load
- Check performance
