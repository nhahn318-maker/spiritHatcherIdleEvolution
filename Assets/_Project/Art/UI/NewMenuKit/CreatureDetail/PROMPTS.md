# CreatureDetail UI Image Prompts

Use these prompts to generate each asset as a separate high-resolution PNG. Keep text out of the images; use TextMeshPro in Unity.

## creature_info_panel.png

```text
Asset group: CreatureDetail
Asset name: creature_info_panel
Create a clean fantasy mobile game UI panel asset, no text, transparent background.
Shape: wide rounded rectangle info card, 608x208 ratio.
Style: polished 2D game UI, warm dark brown translucent body, subtle amber/gold border, soft inner highlight, minimal decoration.
Use: creature detail/info card with empty space for creature portrait on the left and stats on the right.
Constraints: transparent background, crisp edges, no shadow outside the asset, no portrait, no icons, no text, no watermark, no extra ornaments.
```
