# DailyReward UI Image Prompts

Use these prompts to generate each asset as a separate high-resolution PNG. Keep timer, labels, and amounts as TextMeshPro in Unity.

## offline_reward_panel.png

```text
Asset group: DailyReward
Asset name: offline_reward_panel
Create a compact offline reward panel UI asset, transparent background.
Shape: small rounded square or vertical card, no text.
Style: clean fantasy mobile game UI, dark warm brown panel, thin amber/gold border, subtle bevel, simple readable surface.
Use: container for offline reward chest icon and timer text.
Constraints: no chest, no text, no icons, no watermark, transparent outside, crisp edges, 9-slice friendly.
```

## offline_chest_icon.png

```text
Asset group: DailyReward
Asset name: offline_chest_icon
Create a treasure chest reward icon for a fantasy idle mobile game UI, transparent background.
Subject: small closed treasure chest, front view, readable at small size.
Style: polished 2D game icon, orange-brown wood, gold bands, small lock, clean bevels, minimal glow.
Use: offline reward and daily reward button icon.
Constraints: no panel, no text, no coins spilling out, no watermark, transparent outside, crisp silhouette.
```

## notification_dot.png

```text
Asset group: DailyReward
Asset name: notification_dot
Create a small red notification dot UI asset, transparent background.
Shape: circular red badge with tiny highlight and thin gold or light rim.
Style: clean mobile game UI, readable at very small size.
Use: notification marker for available reward.
Constraints: no number, no text, no watermark, transparent outside, crisp circular edge.
```
